    Tecnologias

    Durante a faculdade tive a oportunidade de introdução a algumas tecnologias, tais como PHP, Java, Javascript, SQL. No entanto isso foi uma pequena prévia de como seriam as linguagens de programação e a própria lógica de programação. Durante meu estágio enquanto estava em formação, aprendi um pouco mais sobre SQL Server e também conheci o desenvolvimento RPAs. Após sair da faculdade, ja tinha decidido que iria focar em desenvolvimento Front-End então atualmente estou focado em estudar tecnologias de desenvolvimento Web, a princípio estou estudando e praticando ReactJS, TypeScript e NodeJS.

    Experiências

    Atualmente eu tenho uma unica experiência no mercado de teconologia, no qual trabalhei como estágiario de uma empresa chamada BC FOX Tecnologias, nessa empresa eu realizava tarefas como HelpDesk, realizava desenvolvimento de RPAs, criação de formulários em HTML, CSS, JS e consultas à banco de dados SQL Server.

    Fora o trabalho, realizei algumas maratonas devs que foram produzidas e guiadas pela RocketSeat, no qual tive o desafio de acompanhar o desenvolvimento de uma plataforma inteira do zero, utilizando as stacks ReactJs, React Native e NodeJS.
